Introduction

What this is

  • An illustrated and opinionated HowTo on how I prefer to set up RHODS environments

What this is NOT

This is NOT:

  • A replacement for the entire and complete documentation

  • A way to build production-grade environments

  • Going to be the best design for all situations

Timing

  • Going through these steps will take anywhere between 30 minutes and a few hours

    • The first time is going to take the longest

    • If you already have your cluster ready, it will be a lot faster
